Que es HTML?

Lenguaje de marcado que estructura el contenido de una pagina web. Basado en etiquetas que pueden ser semanticas o no semanticas

Semantica html

Etiquetas basicas HMLT

Etiqueta Descripcion
h1, h2, h3, h4, h5, h56 Titulo y subtitulos HTML
"p" parrafos
img Para mostrar las imagenes (src,alt)
ul, ol Etiqueta para definir listas ordenadas y desordenadas
Div Divisores de contenido para dar estilo
(a) Es redireccionar a un link fuera o dentro de nuestras paginas

formularios HTML

  • Lista desplegables
